## Этапы создание MVP


### Первый Этап

1) n8n как каркас для ИИ Системы/и Пайплайна

2) интеграций и единый Gateway для разных платформ и источников самих запросов Тикетов (в нашем случае это: ), (Приём запросов из почты, чатов, портала, телефонии)

3) backend service: локальная LLM (Ollama, Qwen2.5-1.5B), векторная база данных (Qdrant), и embedding-модель для генерации векторных представлений текста (nomic-embed-text)

4) backend service: для распределние и маршрутизаций для человека который свободен (например есть 3 специалиста доступных как Help Desk которые могут помогать людям и их запросам/Тикетам и наш бэкенд должен давать Человека который имеет статус: Свободен)
